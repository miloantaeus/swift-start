# Install & Usage Instructions

## For Buyers

1. Download the ZIP file from Gumroad after purchase
2. Unzip and read `SKILL.md` first — this is the full toolkit
3. Follow the 7-Day Plan in order (Day 1–7 sections)
4. Use the 21-Day Outreach Sprint templates one per day
5. Apply the Proposal Templates when prospects respond

## For Deployment

To deploy this product:

```bash
# Package the product
cd ~/gumroad-products/ai-freelancer-swift-start/
zip -r ai-freelancer-swift-start.zip \
  01-upwork-proposal-prompts.md \
  02-fiverr-prompts.md \
  03-bid-strategy-guide.md \
  04-proposal-templates.md \
  INSTRUCTIONS.md \
  SKILL.md

# Upload to Gumroad at https://app.gumroad.com/products/new
# Product name: AI Freelancer Swift Start Toolkit
# Price: $19
# Upload the zip as the digital file
```

## Requirements

- Any text editor to read the .md files
- Internet access to use the AI prompts (works with ChatGPT, Claude, Gemini)
- No coding required
